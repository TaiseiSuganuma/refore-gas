# CLAUDE.md — registry-ocr 固有ルール

GAS 共通ルールはリポジトリルートの [`CLAUDE.md`](../../CLAUDE.md) を参照してください。
このファイルには **registry-ocr 固有** の注意点のみを記載します。

## プロジェクト概要

不動産登記簿 PDF を Google Apps Script の OCR で読み取り、スプレッドシートに転記するシステムです。
GAS の種別は **Sheets コンテナバインド型**（カスタムメニューからダイアログを開いて操作）。

## 使用 OAuth スコープ

| スコープ | 用途 |
|---|---|
| `spreadsheets` | シートへの追記 |
| `drive` | PDF の一時保存・OCR 変換・一時ファイル削除 |
| `documents` | OCR 変換後 Docs のテキスト読み出し |

スコープ追加が必要な場合は理由を説明してユーザーに確認すること（共通ルール参照）。

## 依存している Advanced Service

- **Google Drive API v2**（識別子 `Drive`）が必要です。
- `clasp push` 後に GAS エディタの「サービス」から有効化する必要があるため、
  新規環境にセットアップする際は README の手順を必ず参照してください。

## OCR パーサーについて

- 正規表現パーサー: `src/services/registryParser.ts`
- OCR 結果のレイアウトはスキャン品質や登記所によってばらつくため、
  取得できなかった項目は「（取得できませんでした）」と表示します。
- パーサー修正時はサンプル PDF で複数パターン確認してから push してください。

## トリガー

このプロジェクトはトリガー不要（手動実行のみ）です。
将来バッチ処理を追加する場合は GAS エディタの「トリガー」画面から設定します。
