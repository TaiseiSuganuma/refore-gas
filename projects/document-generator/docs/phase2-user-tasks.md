## 作業1: 「案件一覧」シートの列拡張

### 概要

現在の「案件一覧」シート（A〜AN 列）の**右端に新しい列を追加**します。
既存データは一切変更しません。

### 追加する列（AO 列〜）

既存の最後列の右隣から、以下の順番でヘッダーを貼り付けてください。

```
選択	案件パターン	ステータス	フォルダURL	最終出力日時	書類: 土地売買契約書	書類: 立木付土地売買契約書	書類: 立木売買契約書	書類: お客様契約書_土地込	書類: お客様契約書_立木	書類: 御見積書	書類: 請求書	書類: 売上計算シート	書類: チェックリスト	書類: 法務局_X1	書類: 法務局_X2	書類: 法務局_X3	書類: 法務局_X4	書類: 法務局_X5
```

### 列ごとの設定

| 列 | カラム名 | 設定 |
|---|---|---|
| `選択` | チェックボックス | 「書類発行」メニュー実行時の対象選択。**セルを選択 → 挿入 → チェックボックス** で設定 |
| `案件パターン` | プルダウン | データ検証で `A①,A②,A③,B①,B②` の5択に制限 |
| `ステータス` | プルダウン | データ検証で `未着手,出力済み,完了,保留` の4択に制限。既存行は「未着手」を手入力 |
| `フォルダURL` | テキスト | GAS が自動でセットするため空欄のまま |
| `最終出力日時` | テキスト | GAS が自動でセットするため空欄のまま |
| `書類: 〜` × 14列 | チェックボックス | すべてチェックボックスとして設定。初期値はすべて OFF（FALSE） |

### サンプル案件 `EST-20260309-001` への初期値入力

| カラム名 | 入力値 |
|---|---|
| `選択` | FALSE（チェックなし） |
| `案件パターン` | `A③` |
| `ステータス` | `未着手` |
| `書類: 土地売買契約書` | TRUE（チェックあり） |
| その他の書類列 | FALSE |
