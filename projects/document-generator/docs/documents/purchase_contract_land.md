## 既知の制約・特記事項

### テンプレ準備が必須

- 現状のテンプレには **プレースホルダがまだ入っていない**（`____` のみ）
- Phase 1 着手時、ユーザーがテンプレに `{{key}}` を挿入する必要がある
- テンプレ編集後はコピーを取り、原本に手を加える前に動作確認する

### マスタシートに不足する情報

- 第１条の「地番」「地目」「筆数」 → 案件一覧に該当列が無い。登記情報を別途取り込むか、入力列を追加するか要決定
- 第２条「手付金」 → 案件一覧に列無し（金額の一部として手付金がいくらかは別管理が必要）
- 第３条「支払予定日」 → 入力列を追加するか、ダイアログで入力させるか

### Docs の `replaceText` 注意点

- Google Docs は文字単位で書式が異なると `body.replaceText("{{key}}", ...)` がマッチしない
- テンプレ作成時、プレースホルダは **同じフォント・書式で一気に入力** する
