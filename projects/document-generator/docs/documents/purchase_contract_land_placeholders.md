# 土地売買契約書 — プレースホルダ挿入案

「土地売買契約書」テンプレ（[Docs URL](https://docs.google.com/document/d/1AsZLK9_rmIF-VC62I42QMAmPmzNz0kp1KmegOZ6ApJI/edit)）に対する
Phase 1 でのプレースホルダ挿入案。

## 作業の進め方

1. **テンプレ原本のコピーを取る**（Drive 上で「コピーを作成」）
2. コピーに対して、本ファイルに示す箇所に `{{ }}` で囲んだ文字列を **手で書き込む**
3. プレースホルダは半角の波括弧2つ `{{` `}}` を使う
4. **重要**: 1つのプレースホルダは **同じ書式設定で一気に入力** すること（Docs の仕様上、文字単位で書式が異なると置換できない）
5. 書き込み後、コピー側の Docs ID を control.gas または書類マスタに登録

> **注意**: 原本テンプレに直接書き込まず、コピーで運用すること。
> 万一プレースホルダの仕様変更があった場合に原本に戻れる保険として。

## プレースホルダ一覧（差込前 → 差込後）

### 文末の契約締結日と甲（売主）情報

#### 契約締結日

```
変更前: 　　　　　　　　年　　　月　　　日
変更後: {{契約日}}
```

差込例: `令和8年5月17日`

#### 甲（売主）情報

```
変更前:
甲　　（ご本人：又は　　　　　　　　様名義の続柄　　　　　）

住所　　　　　　　　　　　　　　　　　　　　　　　　

氏名　　　　　　　　　　　　　　　　　　　　　　　印

連絡先　　　　　　　　　　　　　　　　　　　　　　　　

変更後:
甲　　（ご本人：又は{{売主名義人}}様名義の続柄{{売主続柄}}）

住所　{{売主住所}}

氏名　{{売主氏名}}　　　　　　　　　　　　　　　印

連絡先　{{売主連絡先}}
```

差込例:
- 売主名義人: `佐藤てるお`
- 売主続柄: 本人の場合は空文字 → `（ご本人：又は佐藤てるお様名義の続柄）`
- 売主住所: `鹿児島県鹿屋市西祓川町509番地5`
- 売主氏名: `佐藤てるお`
- 売主連絡先: `080-xxxx-xxxx`

> **乙（リフォレ）情報は固定**（Phase 1 では変更しない）
