# 立木付土地売買契約書 — プレースホルダ挿入案

A② パターン（立木+土地をセットで売買し、金額の内訳を厳密に分ける）の契約書テンプレ
（[【テンプレ】立木付土地売買契約書](https://docs.google.com/document/d/1jZn6tVw7VD_QhWj9PkUolVoNiL_HM81VPRlv_KWkKHg/edit?usp=drive_link)）
に対するプレースホルダ挿入案。

土地売買契約書（A③）との違い: **第１条第２項の対価が「土地代金（非課税）」と「立木代金（消費税10%）」の2つに分かれる**。
他の条文は基本的に同じ。

## 作業の進め方

土地売買契約書と同じ。[`purchase_contract_land_placeholders.md`](./purchase_contract_land_placeholders.md) の「作業の進め方」セクション参照。

#### 第１条 第２項（金額・内訳）

A② 固有。土地代と立木代を分けて記載する。

```
変更前:
２　本件物件の対価として、乙は甲に対し、合計金　　　　　　円（消費税込）を支払う。
1.  土地代金　　　金　　　　　　円（非課税）
2.  立木代金　　　金　　　　　　円（消費税10％）

変更後:
２　本件物件の対価として、乙は甲に対し、合計金{{売買金額合計}}円（消費税込）を支払う。
1.  土地代金　　　金{{土地代金}}円（非課税）
2.  立木代金　　　金{{立木代金}}円（消費税10％）
```

差込例:
- 売買金額合計: `1,100,000`（土地500,000 + 立木600,000 × 1.1）
- 土地代金: `500,000`
- 立木代金: `545,454`（税抜）

> **計算ルール（GAS 側）**:
> - 立木代金は税抜表記。消費税は契約書本文の「（消費税10％）」で表現されているため、税抜金額を入れる
> - 売買金額合計は「土地代金（非課税）+ 立木代金（税込）」= 土地代金 + 立木代金 × 1.1
> - 金額の根拠は物件シート（樹種別・物件別の金額を集計）から算出。詳細ルールは Phase 3 着手時に確定
