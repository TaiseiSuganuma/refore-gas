# 書類ごとの仕様

このディレクトリには、生成する各書類の個別仕様を 1 ファイル / 1 書類で記述する。

## ファイル命名

`<書類ID>.md`（半角英小文字スネークケース）
例: `sales_contract.md`, `important_matter_explanation.md`

## 各書類仕様書のテンプレート

新しい書類を追加するときは [`_template.md`](./_template.md) をコピーして埋める。
