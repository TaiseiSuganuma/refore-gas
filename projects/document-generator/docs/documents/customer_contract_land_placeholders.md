# お客様契約書_土地込 — プレースホルダ挿入案

B② パターン（リフォレ→買主、土地と山林をセットで販売）の契約書テンプレ
（[【テンプレ】売買契約書_土地込](https://docs.google.com/document/d/1iSftDh4XjSVklC9osdOaoWvOKcKH_6ZATNuNzHPoToU/edit?usp=drive_link)）
に対するプレースホルダ挿入案。

A 系統との大きな違い:
- **売主 = リフォレ（固定）、買主 = 取引先マスタ参照**
- 簡易フォーマット（1 ページ程度）。条文番号 1〜4 のみ
- **別紙に物件一覧（地番）を繰返し記述** → Phase 3 の繰返し処理対象

## 作業の進め方

土地売買契約書と同じ。[`purchase_contract_land_placeholders.md`](./purchase_contract_land_placeholders.md) の「作業の進め方」セクション参照。

## プレースホルダ一覧（差込前 → 差込後）

### 第１条相当（物件・面積・金額）

```
変更前:
1.  物件所在地：　　　　　　　　　　（　　　　　㎡）　他　筆　※詳細は別紙にて記載

1.面積：　筆　合計　　　　　㎡（雑抜き:　　　㎡）

1. 金額合計：　　　　　　円   （消費税　　　　　円）
◇山林：　　　　　円　（消費税　　　　　円）10％対象
◇土地：　　　　　円　（消費税―――――円）　非課税

変更後:
1.  物件所在地：{{代表所在}}（{{代表面積}}㎡）他 {{他筆数}}筆 ※詳細は別紙にて記載

1.面積：{{筆数}}筆 合計{{合計面積}}㎡（雑抜き:{{合計雑抜き}}㎡）

1. 金額合計：{{売買金額合計}}円（消費税{{消費税合計}}円）
◇山林：{{山林金額}}円（消費税{{山林消費税}}円）10％対象
◇土地：{{土地金額}}円（消費税―――――円）　非課税
```

差込例:
- 代表所在: `南さつま市加世田内山田10647-1`（物件シート 1 行目を代表に）
- 代表面積: `7,986`
- 他筆数: `2`（全 3 筆なら「他 2 筆」）
- 筆数: `3`
- 合計面積: `11,326`
- 合計雑抜き: `4,403`
- 売買金額合計: `1,100,000`
- 消費税合計: `54,545`
- 山林金額: `545,454` / 山林消費税: `54,545`
- 土地金額: `500,000`

> **計算ルール（GAS 側）**: 物件シートを案件 ID でフィルタ → 集計。Phase 3 で実装。

#### 売買代金（再掲）

```
変更前: 2.売買代金は　金　　　　　円(手数料　消費税込み)と定め、買主は売主に支払うものとする。
変更後: 2.売買代金は　金{{売買金額合計}}円(手数料　消費税込み)と定め、買主は売主に支払うものとする。
```

### 契約日

```
変更前: 2023年　　月　　　日
変更後: {{契約日}}
```

差込例: `令和8年5月17日` / `2026年5月17日`

　　　　氏名：
　　　　住所：
         登録番号：

変更後:

### 別紙詳細（地番リスト） — Phase 3 で繰返し処理

```
変更前:
〇別紙詳細
【地番】
1.    
2.    
　　以上　　筆

変更後（Phase 3 の繰返し記法）:
〇別紙詳細
【地番】
{{#each items}}
{{物件No}}.  {{所在}}{{地番}}
{{/each}}
　　以上 {{筆数}}筆
```

> **Phase 3 で対応**: 物件シートを案件 ID で抽出して繰返し展開する処理。
> Phase 2 では「別紙は手作業で追記」運用にしてもよい（要ユーザー判断）。
