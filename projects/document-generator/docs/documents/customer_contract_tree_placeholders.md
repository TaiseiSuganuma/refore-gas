- レイアウトはほぼ B② と同じ簡易フォーマット

## プレースホルダ一覧（差込前 → 差込後）

### 物件情報 〜 金額

B② と完全に同じ構造。

```
変更後:
1.  物件所在地：{{代表所在}}（{{代表面積}}㎡）他 {{他筆数}}筆 ※詳細は別紙にて記載

1.面積：{{筆数}}筆 合計{{合計面積}}㎡（雑抜き:{{合計雑抜き}}㎡）

1. 金額合計：{{売買金額合計}}円（消費税{{消費税合計}}円）
◇山林：{{山林金額}}円（消費税{{山林消費税}}円）10％対象
◇土地：{{土地金額}}円（消費税―――――円）　非課税
```

> **注**: テンプレ名が「土地別」だが、金額表記は「山林」「土地」を分けるレイアウトのまま。実運用では土地金額が 0 円のケースがほとんどになる想定。

```
変更前: 2.売買代金は　金　　　　　円(手数料　消費税込み)と定め、買主は売主に支払うものとする。
変更後: 2.売買代金は　金{{売買金額合計}}円(手数料　消費税込み)と定め、買主は売主に支払うものとする。
```

### 買主情報 — **取引先マスタから引き直す**

**【テンプレ】には現状「株式会社 陽 / 加塩 忠勇 / 鹿児島県志布志市志布志町帖3946-8 / T6340001015461」が印字されているが、案件によって買主が変わる場合に備え、プレースホルダ化する**。

```
変更前:
買主　　株式会社　陽
　　　　氏名：加塩　忠勇
　　　　住所：鹿児島県志布志市志布志町帖3946-8
         登録番号：T 6340001015461

変更後:
買主　　{{買主会社名}}
　　　　氏名：{{買主代表者名}}
　　　　住所：{{買主住所}}
         登録番号：{{買主登録番号}}
```

差込例（取引先マスタから引いた場合）:
- 通常パターン: 株式会社 陽 / 加塩 忠勇 / 志布志市... / T6340001015461
- 別業者パターン: 案件マスタの `買主取引先ID` に別の取引先 ID を指定すれば自動で差し替わる
