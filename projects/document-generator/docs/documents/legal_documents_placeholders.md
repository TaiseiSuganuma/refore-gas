# 法務局関連 5 書類 — プレースホルダ挿入案

土地が絡む契約（A②/A③/B②）で法務局に提出する 5 書類。
書類数が多いが、登場する情報のセットが似ているため 1 ファイルにまとめる。

差込例:
- 住所変更日_和暦: 売主が住所を変更した日（案件マスタに専用列追加が必要）

## プレースホルダ一覧（サマリー）

法務局 5 書類で共通的に登場するプレースホルダ:

| プレースホルダ | 説明 | データ源 |
|---|---|---|
| `{{契約日_和暦}}` | 売買契約日 | 案件マスタ「契約日」を和暦変換 |
| `{{書類作成日_和暦}}` | 申請日 | 生成日または案件マスタの新規列 |
| `{{売主氏名}}` `{{売主住所}}` `{{売主連絡先}}` `{{売主生年月日}}` | 売主（義務者・委任者） | 案件マスタ |
| `{{売主新住所}}` `{{売主旧住所}}` `{{住所変更日_和暦}}` | 住所変更系（住所変更書類のみ） | **案件マスタへ列追加が必要** |
| `{{買主会社名}}` `{{買主住所}}` | 買主（権利者） | 取引先マスタ |
| `{{代理人氏名}}` `{{代理人住所}}` | 代理人 | **代理人マスタ** |
| `{{法務局支局}}` | 提出先支局 | 案件マスタの新規列 or 設定シート（地域から判定） |
| `{{不動産表示_全筆}}` | 不動産の表示（簡易版） | 物件シート集計 |
| `{{不動産番号}}` `{{所在}}` `{{地番}}` `{{地目}}` `{{地積}}` | 不動産の表示（繰返し） | 物件シート（**`不動産番号` 列追加が必要**） |
| `{{課税価格}}` `{{登録免許税}}` `{{登録免許税_住所変更}}` | 税金 | GAS 計算 |

## 各シートへの追加が必要なもの（Phase 3 着手時に確定）

### 案件マスタへの列追加

- `売主生年月日` — 登記申請書で必要
- `売主旧住所` `売主新住所` `住所変更日` — 住所変更書類で必要（住所変更が無い案件では空）
- `法務局支局` — 提出先支局名（プルダウンで「加世田」「鹿屋」等）

### 物件シートへの列追加

- `不動産番号` — 登記簿の 12〜13 桁の番号。registry-ocr で抽出済みの可能性あり（Phase 5 で連携）

### 代理人マスタ（[`../master-sheet-schema.md` § 10](../master-sheet-schema.md#10-代理人マスタシートphase-2-新設) で新設）

代理人氏名・住所をここから引く。
