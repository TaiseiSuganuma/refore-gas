# 立木売買契約書 — プレースホルダ挿入案

A① パターン（立木のみを売買、土地は対象外）の契約書テンプレ
（[【テンプレ】立木売買契約書](https://docs.google.com/document/d/10oZ_KmhB0N8oppBu-Z2jW0eAYk0mRdIZORIJn2yXE7I/edit?usp=drive_link)）
に対するプレースホルダ挿入案。

土地売買契約書（A③）との違い:
- 第１条第２項の対価が **単一金額（消費税込）**
- 第７条に **立木の伐採・搬出条項** が追加されている（プレースホルダ不要 = 固定条文）
- 公租公課・保証・所有権移転先指定の条項が **無い**（立木のみのため）

## 作業の進め方

土地売買契約書と同じ。[`purchase_contract_land_placeholders.md`](./purchase_contract_land_placeholders.md) の「作業の進め方」セクション参照。

### 第１条（本件立木の売買）

#### 第１条 第１項（土地表記）

立木の所在地となる土地を記述する。土地売買契約書と同一形式。

```
変更前:
記
1.  土地
所　在　　　　　　
　　　地　番　　　　　　
　　　地　目　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　筆　数　　　　　
　　　地　積　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　合　計
1.  立木
      1.  記載の土地上に存在するすべての立木

変更後:
記
1.  土地
所　在　{{所在}}
　　　地　番　{{地番}}
　　　地　目　{{地目}}　　　　　　　　　　　　　　　　　　　　　　　　　　　筆　数　{{筆数}}
　　　地　積　{{地積}}　　　　　　　　　　　　　　　　　　　　　　　　　　　合　計　{{土地合計}}
1.  立木
      1.  記載の土地上に存在するすべての立木
```

#### 第１条 第２項（金額）

A① 固有。立木のみ単一金額。

```
変更前:
２　本件立木の対価として、乙は甲に対し、金　　　　　　円（消費税込）を支払う。

変更後:
２　本件立木の対価として、乙は甲に対し、金{{売買金額}}円（消費税込）を支払う。
```

差込例: `600,000`（消費税込）

> **注**: A③ 土地売買契約書は「消費税別」、A① 立木売買契約書は「消費税込」と本文表記が違うので、`{{売買金額}}` に入れる値が違う。Phase 3 着手時に GAS の金額整形ロジックでこの違いを吸収する。

### 第２条（手付金）

土地売買契約書と同一。

```
変更前: １　乙は、本契約時に手付金として金　　　　　　円を支払う。
変更後: １　乙は、本契約時に手付金として金{{手付金}}円を支払う。
```

土地売買契約書と同一。

```
　　口座番号：{{口座番号}}
　　口座名義人：{{口座名義}}　　フリガナ：{{口座名義カナ}}
```

---

### 第３条（代金の支払い）

土地売買契約書と同一。

```
変更後:
{{支払予定日}}限り現金一括支払い
{{支払予定日}}限り振込一括支払い
```

---

### 第７条（立木の伐採） — プレースホルダ不要

A① 固有の条項（伐採時期・隣接土地使用）。**現状のテンプレに具体的な日付・期間は入っていない**ため、プレースホルダ化しない。
案件ごとに伐採時期を契約書に書き込む必要が出てきた場合は Phase 3 以降で `{{伐採開始日}}` 等を追加検討する。
