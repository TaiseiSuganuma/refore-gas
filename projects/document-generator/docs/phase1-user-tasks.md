# Phase 1 ユーザー作業チェックリスト

Phase 1 の GAS 実装に進む前に、ユーザー側で手作業で実施すべき項目をまとめます。
すべて完了してから [`#4 Phase 1 MVP の GAS 実装`] に進みます。

> **位置づけ**: Phase 1 は **土地売買契約書 1 書類だけを単発で生成する MVP**。
> 本格的なバッチ型・集約型運用（複数案件×複数書類の一括出力、書類列チェックボックス、パターン判定）は
> **Phase 2 で実装**します。本書のユーザー作業は MVP のための最小準備で、
> Phase 2 着手時にマスタ列の大幅追加（書類選択列・選択チェック列・案件パターン列など）が改めて発生します。
> 詳細は [`specification.md` § 3 Phase 設計](./specification.md#3-段階的アプローチphase-設計) 参照。

## 作業1: 「土地売買契約書」テンプレのコピーとプレースホルダ挿入

### 手順

1. 原本テンプレ（[土地売買契約書](https://docs.google.com/document/d/1AsZLK9_rmIF-VC62I42QMAmPmzNz0kp1KmegOZ6ApJI/edit)）を Drive で開く
2. メニュー「ファイル」→「コピーを作成」
3. コピーの名前を `テンプレ_土地売買契約書` などに変更（運用者がわかる名前）
4. コピー側を開く
5. [`documents/purchase_contract_land_placeholders.md`](./documents/purchase_contract_land_placeholders.md) の「変更前 → 変更後」を参考に、`{{ }}` プレースホルダを順次書き換える
6. **重要**: 1つの `{{xxx}}` は途切れず同じ書式で入力する（途中で書式変更があると GAS の置換でマッチしないため）
7. 書き換え完了後、コピーの URL（Docs ID）をメモ

### 完了報告に含めて欲しい情報

- コピー版テンプレの Docs URL（または fileId）

## 作業2: 「案件一覧」シートに 6 列追加

### 手順

1. [【リフォレ】書類自動生成](https://docs.google.com/spreadsheets/d/12BEUxTqnADqWsmffGWos0tFe08FohEE-0RQJbLNAnEk/edit) を開く
2. 「案件一覧」シートの **S1 セル以降** に以下のヘッダーをタブ区切りで貼り付ける

```
契約日	売主名義人	売主続柄	手付金	口座名義カナ	支払予定日
```

3. 既存のテストデータ（`EST-20260309-001` など）について、新規列にもサンプル値を入れる:
   - `契約日`: `2026/06/15`（例）
   - `売主名義人`: 空（本人と同じ場合）
   - `売主続柄`: 空
   - `手付金`: `50000`
   - `口座名義カナ`: `サトウテルオ`
   - `支払予定日`: `2026/07/15`

## 作業3: 「明細」シートを「物件」シートに再設計

### 手順

1. シートタブ「明細」を **「物件」** にリネーム
2. **A1 セル以降** に新ヘッダーをタブ区切りで貼り付ける（既存ヘッダーは置き換え）

```
案件ID	物件No	所在	地番	地目	面積(㎡)	雑抜き(㎡)	樹種	樹高(m)	金額(円)	備考
```

3. 既存データを新形式に整理（参考: [`phase1-sheet-changes.md`](./phase1-sheet-changes.md)）:
   - 旧「住所（地目）」を「所在」「地番」に分割
   - 「`〃`」表記は廃止し、すべての行に所在を明記
   - 「地目」列には登記情報の地目（`山林` 等）を入れる
   - 元の `（立木）` 表記は契約区分なので物件シートには入れない

## 作業5: Phase 1 用テストフォルダの作成

### 手順

1. Drive で **【リフォレ】書類自動生成スプレッドシートと同じ親フォルダ** を開く
2. その中に新規フォルダ `01_Phase1動作確認用` を作成
3. 作ったフォルダを開き、URL から fileId をコピー
   - URL の `folders/` の後ろの英数字部分
   - 例: `https://drive.google.com/drive/folders/1AbCdEfGhIjKlMnOpQrStUv` → `1AbCdEfGhIjKlMnOpQrStUv`
4. 「設定」シートの `出力先フォルダID` に貼り付け
